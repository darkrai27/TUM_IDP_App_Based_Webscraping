users.queryuser(arg: int, dtsg: str, session_id: str) → Json
users.queryuser(arg: str, dtsg: str, session_id: str) → Json

Returns a user profile by its user internal id (number) or it’s username (string) is matched

get_posts

users.get_posts(username, dtsg, session_id)

Returns most recent posts of a user by its internal user id

scrap_all_posts

users.scrap_all_posts(username: str, dtsg: str, session_id: str, limit: int = None, delay: int = 5)

Returns all posts of the account with the passed username using its internal user_id or the most recent up post to a limit of posts. Each query is performed each delay seconds (5 by default) to avoid banning the account for bot activity.

get_follows_info

users.get_follows_info(username: str, dtsg: str, session_id: str)

Returns an overview information of the number of followers and following of an account
